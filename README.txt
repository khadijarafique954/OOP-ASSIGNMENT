Project Title:
Ride Fare Calculation & Trip Management Library

Description:
This project is a C++ OOP-based library that calculates ride fares and manages trip data.
It demonstrates function overloading for multiple fare calculation methods and operator overloading
for combining rides, comparing fares, and applying surge pricing.

Key Features:
- Constructor overloading
- Function overloading (fare calculation)
- Operator overloading (+, -, *, ==, <, <<)
- Object-oriented design

Sample Output:

Ride Fare: 0
Ride Fare: 255
